### عن المشروع
نظام إدارة متكامل (POS + CRM) مبني بتقنيات الويب، يدعم:
- إدارة المبيعات والفواتير
- إدارة المخزون
- العملاء والموردين
- تقارير مالية
- طباعة الفواتير
- دعم Firebase (قاعدة بيانات سحابية)

### المميزات المتقدمة
- واجهة Liquid Glass Design حديثة
- أكثر من ثيم (Glass, Light, Night, Neumorphism)
- دعم كامل للغة العربية (RTL)
- تكامل مع Firebase Auth + Firestore
- تصدير PDF + QR Code
- لوحة تحكم إدارية

### التقنيات المستخدمة
- HTML5 + Tailwind CSS
- Vanilla JavaScript + React (UMD)
- Firebase (Firestore + Auth)
- html2pdf + QRious
